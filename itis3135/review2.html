Screenshot of the Site

Evaluation Checklist

  • File/Folder Naming: All file and folder names are lowercase, with no spaces though underscores are used.
  • Design: The page has somewhat good contrast, appropriate font sizes, and uses a standard `.css` file.
  • CRAP Principles:
    • Contrast: Text and background colors provide clear contrast though I would avoid using green on green.
    • Repetition: Consistent use of colors, fonts. However, the layout is not consistent throughout the site.
    • Alignment: Elements are not entirely properly aligned, on some pages the text is centered and on others it is left indent.
    • Proximity: Related elements are grouped together.
  • Page Structure:
    • Header includes a site/brand header with text in an h1 tag.
    • Main starts with the name of the page as an h2 tag.
    • Footer includes a navigation menu and “Designed by…” text.
  • Validation: The page does not pass HTML and CSS validation.
  • Assignment-Specific Requirements:Not all requirements for the assignment are met.

Additional Observations

  • The Good: The site is clean and has a refreshing feel to it.
  • The Bad: The visuals need to be more consistent and adding more content could help.
  • The Ugly: As far as I can tell there is only one form of interactivtiy and the page structure must be addressed before submission.

Suggestions

  • Start: Adding more content and addressing the structural erorrs as well as adding more interactivity.
  • Continue: Keep the theme of the website going, the scheme is nice.

Conclusion

My overall impression of the site is that I like the direction its headed but it needs to address some very critical errors before submission.
