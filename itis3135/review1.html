Webpage Review #1

Name of the Person Being Evaluated:

Crosby, Ryan

Screenshot of the Site

Evaluation Checklist

  • File/Folder Naming: All file and folder names are lowercase, with no spaces.
  • Design: The page has sufficient contrast, appropriate font sizes, and uses a standard `.css` file but the may be too wordy.
  • CRAP Principles:
    • Contrast: Text and background colors provide clear contrast. However could improve upon image sizing,
    • Repetition: Consistent use of colors, fonts, and layout.
    • Alignment: Elements are properly aligned.
    • Proximity: Related elements are grouped together.
  • Page Structure:
    • Header includes a site/brand header with text in an h1.
    • Main starts with the name of the page as an h2.
    • Footer includes a navigation menu and “Designed by…” text.
  • Validation: The page passes HTML and CSS validation.
  • Assignment-Specific Requirements: All requirements for the assignment are met.

Additional Observations

  • The Good: The color scheme matches the theme and it is an easy to navigate site.
  • The Bad: The pictures are too large and it feels like there is too much text.
  • The Ugly: Missing a third form of interactivty.

Suggestions

  • Stop: Reduce overall image size and text length.
  • Start: Using consistent image sizes..
  • Continue: Keep delivering quality information and keep up with the theme. This project is going to be good!

Conclusion

I think that overall this is a strong website and that this person will receive a high grade. They just need to address the visual issues as well as adding the third form of interactivity.
